# 💖 SheShield AI – Women Safety Companion

SheShield AI is a smart AI-powered safety assistant built to help users stay safe during travel, emergencies, and daily life. It includes an AI chatbot, SOS alerts, and safety guidance features.

---

## 🚀 Live Demo

👉 [https://sheshield-ai-uqpv.vercel.app/](https://sheshield-ai-uqpv.vercel.app/)

## ✨ Features

### 🛡️ AI Safety Chat

* Ask safety-related questions
* Get instant guidance
* Emotional & supportive AI responses

### 🚨 SOS System (UI Ready)

* Emergency alert concept
* One-tap safety trigger (expandable)

## ⚙️ Setup Instructions

### 1. Clone Repo

```bash
git clone https://github.com/your-username/sheshield.git
cd sheshield
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Add Environment Variables

Create `.env.local`

```env
GROQ_API_KEY=your_key_here
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
```

### 4. Run Locally

```bash
npm run dev
```
